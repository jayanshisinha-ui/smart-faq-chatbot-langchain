# 🤖 Smart FAQ Chatbot (LangChain)

An offline ChatGPT-style FAQ chatbot built using **LangChain, FAISS, HuggingFace Embeddings, and Tkinter GUI** with memory support for intelligent question answering.

---

# 📌 Project Overview

Smart FAQ Chatbot is an AI-based desktop chatbot that answers user questions from stored knowledge using vector similarity search.

The chatbot:

- Uses **LangChain** for processing text
- Uses **FAISS** for vector storage
- Uses **HuggingFace embeddings**
- Provides a **ChatGPT-style GUI**
- Saves chat history automatically
- Works **completely offline**

Wait a few seconds while the model loads.

# 🧠 How It Works

1. Loads FAQ dataset  
2. Splits text into smaller chunks  
3. Converts text into embeddings  
4. Stores vectors using FAISS  
5. Matches user question  
6. Returns most relevant answer  
7. Saves conversation memory
